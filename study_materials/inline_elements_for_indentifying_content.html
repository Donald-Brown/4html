Inline elements for indentifying content

abbr
- Used for abbreviations
cite
- Used to indicate bibliographic citation like a book title
code
- Used for computer code, displays in monospace font
dfn
Used for special terms that can be defined elswhere
em
- Indicates content is emphasized, displays in italics
kbd
- Used for keyboard entry, displays monospace
q
- Used for quotations, displays with quotes
samp
- Used to mark sequence of characters that has no other meaning
small
- Used to display fine print in smaller text, ex. footnotes
strong
- Indicates content is emphasized, displays bold
var
- Used for computer variables, displays monospace
