jQuery FAQs

What is jQuery?

jQuery is a library of the JavaScript functions that you're mosy likely to need as you develop web sites.

Why is jQuery becoming so popular?

Three reasons:

  • It's Free
  • It helps you get more done in less time
  • All of it's functions are cross-browser compatible

Which is harder to learn: JavaScript or jQuery?

For most functions, jQuery is significantly easier to learn than JavaScript. But remember jQuery is JavaScript.
